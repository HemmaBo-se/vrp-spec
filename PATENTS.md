# Patent commitment — Vacation Rental Protocol specification

The VRP specification text in this repository is dedicated to the public domain
under [CC0 1.0](./LICENSE). CC0 does not grant patent rights. To remove patent
uncertainty for implementers:

Rouiada Abbas and HemmaBo irrevocably commit, on a royalty-free,
non-discriminatory basis, **not to assert** any patent claims they own or
control that are necessarily infringed by a **conforming** implementation of
this specification, against any party making, using, selling, or distributing
such a conforming implementation.

This commitment does not extend to (a) implementations that do not conform to the
specification, or (b) patent claims infringed only by adding features the
specification does not require. It is a non-assertion covenant, not a transfer of
ownership.

> DRAFT — this wording should be read by an IP lawyer before it is relied upon.
> Reviewing it does not block the CC0 license; it can be refined in a follow-up.
