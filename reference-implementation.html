Reference Implementation

VRP is an open protocol. HemmaBo is the first commercial runtime; villaakerlyckan.se is the public proof node.

No central issuer. No registry gatekeeper. Any verifier can fetch discovery, JWKS, and a signed verified_stay_offer from the host-owned domain.
